# 📋 Пошаговая инструкция: Сборка APK через GitHub

## Шаг 1: Создание репозитория на GitHub

1. **Зайди на GitHub:**
   - Открой https://github.com
   - Войди в свой аккаунт (или зарегистрируйся)

2. **Создай новый репозиторий:**
   - Нажми зеленую кнопку "New" или "+"
   - Выбери "New repository"
   - **Название:** `airhockey-android` (или любое другое)
   - **Описание:** `Аэрохоккей игра для Android`
   - **Публичный** (Public) - обязательно!
   - ✅ Add a README file
   - Нажми "Create repository"

## Шаг 2: Загрузка файлов в репозиторий

### Способ A: Через веб-интерфейс (проще)

1. **В созданном репозитории:**
   - Нажми "uploading an existing file"
   - Перетащи все файлы из папки с игрой:
     - `main.py`
     - `android_hockey.py` 
     - `buildozer.spec`
     - `.github/workflows/build-apk.yml`
     - `README_GITHUB.md`

2. **Коммит файлов:**
   - Внизу страницы напиши: "Add game files"
   - Нажми "Commit changes"

### Способ B: Через Git (если умеешь)

```bash
git clone https://github.com/ТВОЙ_ЛОГИН/airhockey-android.git
cd airhockey-android
# Скопируй все файлы игры в эту папку
git add .
git commit -m "Add airhockey game"
git push
```

## Шаг 3: Создание папки .github/workflows

**ВАЖНО:** Если загружал через веб-интерфейс:

1. В репозитории нажми "Create new file"
2. В поле имени файла напиши: `.github/workflows/build-apk.yml`
3. Скопируй содержимое из файла `build-apk.yml`
4. Нажми "Commit new file"

## Шаг 4: Запуск автоматической сборки

1. **Перейди в раздел Actions:**
   - В своем репозитории нажми вкладку "Actions"
   - Увидишь workflow "Build Android APK"

2. **Запусти сборку:**
   - Нажми на "Build Android APK"
   - Справа нажми "Run workflow"
   - Выбери ветку "main" 
   - Нажми зеленую кнопку "Run workflow"

3. **Жди результата:**
   - Сборка займет 10-15 минут
   - Увидишь зеленую галочку ✅ при успехе
   - Красный крестик ❌ при ошибке

## Шаг 5: Скачивание APK

1. **После успешной сборки:**
   - Кликни на завершенный запуск (зеленая галочка)
   - Прокрути вниз до раздела "Artifacts"
   - Скачай `airhockey-debug-apk.zip`

2. **Распакуй архив:**
   - Внутри найдешь файл `*.apk`
   - Это и есть твоя игра для Android!

## Шаг 6: Установка на телефон

1. **Перенеси APK на телефон:**
   - Через USB кабель
   - Через облако (Google Drive, Яндекс.Диск)
   - Через мессенджер самому себе

2. **Разреши установку:**
   - Настройки → Безопасность → Неизвестные источники ✅
   - Или при установке нажми "Разрешить"

3. **Установи игру:**
   - Открой APK файл
   - Нажми "Установить"
   - Готово! 🎉

## 🔧 Если что-то пошло не так

### Ошибка в Actions:
- Проверь, что все файлы загружены правильно
- Убедись, что репозиторий публичный
- Попробуй запустить сборку еще раз

### APK не устанавливается:
- Проверь, что разрешена установка из неизвестных источников
- Убедись, что APK файл не поврежден
- Попробуй перезагрузить телефон

### Нужна помощь:
- Проверь логи в разделе Actions
- Создай Issue в репозитории
- Попробуй другой способ сборки
