# 🏒 Аэрохоккей для Android

Быстрая игра в аэрохоккей для Android устройств, написанная на Python с использованием Kivy.

## 🚀 Как получить APK файл

### Способ 1: Автоматическая сборка через GitHub Actions

1. **Создай репозиторий на GitHub:**
   - Зайди на https://github.com
   - Нажми "New repository"
   - Назови репозиторий, например `airhockey-android`
   - Сделай его публичным
   - Нажми "Create repository"

2. **Загрузи файлы:**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/ТВОЙ_ЛОГИН/airhockey-android.git
   git push -u origin main
   ```

3. **Запусти сборку:**
   - Перейди в раздел "Actions" в своем репозитории
   - Нажми "Build Android APK"
   - Нажми "Run workflow" → "Run workflow"
   - Жди 10-15 минут

4. **Скачай APK:**
   - После завершения сборки перейди в "Actions"
   - Кликни на последний успешный запуск
   - Скачай файл `airhockey-debug-apk`

### Способ 2: Готовый APK (если доступен)

Если в репозитории есть раздел "Releases", скачай последний APK файл оттуда.

## 🎮 Особенности игры

- **Управление:** Перетаскивай синюю ракетку пальцем
- **Цель:** Забей 7 голов первым
- **ИИ:** Умный противник с разными уровнями сложности
- **Физика:** Реалистичная физика движения шайбы
- **Графика:** Яркий неоновый стиль
- **Ориентация:** Поддержка портретной и альбомной ориентации

## 📱 Установка APK

1. Скачай APK файл
2. На Android устройстве разреши установку из неизвестных источников:
   - Настройки → Безопасность → Неизвестные источники
3. Открой скачанный APK файл
4. Нажми "Установить"

## 🛠️ Для разработчиков

### Структура проекта:
- `main.py` - Точка входа для buildozer
- `android_hockey.py` - Основной код игры
- `buildozer.spec` - Конфигурация для сборки APK
- `.github/workflows/build-apk.yml` - GitHub Actions для автосборки

### Локальная сборка (Linux/macOS):
```bash
pip install kivy buildozer
buildozer android debug
```

### Требования:
- Python 3.8+
- Kivy 2.2.0
- Buildozer 1.5.0
